About Me

Halo! Saya Kusindra Aji Rabbany, atau biasa dipanggil Ben. Saya adalah Information Technology undergraduate di Universitas Brawijaya dengan pengalaman hands-on dalam pengembangan aplikasi web dan mobile.

Selama kuliah dan karir saya, saya telah memimpin berbagai proyek teknologi sebagai Web Project Manager dan General Manager di MokletDev. Saya berhasil mengkoordinasikan 5+ proyek pengembangan software dari tahap perencanaan hingga deployment, sambil membimbing tim untuk menghasilkan aplikasi web yang sustainable.

Pengalaman saya sebagai Frontend Developer di Illiyin Studio mengajarkan saya untuk cepat beradaptasi dengan codebase yang kompleks. Saya berhasil deliver 20+ improvements dan fitur baru untuk aplikasi production-grade, dengan evaluasi kerja mencapai 97/100 untuk teamwork dan 95/100 untuk problem-solving.

Saat ini, saya bekerja sebagai Full-stack Developer. Saya passionate menciptakan aplikasi yang memberikan impact nyata bagi pengguna.
